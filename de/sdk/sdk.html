SDK, software development
tools for developers

A software development kit (SDK) is a set of tools that includes fully developed technologies, engines, and more to help you develop applications which integrate with your platforms.

The Hancom SDK modularizes robust technology and expertise, providing a simple set of tools to help you develop and enhance your products.
