AI SDK: OCR

We provide OCR technology as an SDK that recognizes and extracts text from images, PDF files, and more.

The SDK recognizes and processes specified text based on user-defined keywords, classifying and formatting the text accordingly.

  1. OCR

    Image correction
    Text detection/ recognition

  2. Document classification

    Automatic title area detection
    Custom document classification

  3. Data extraction

    Table/form recognition
    Custom data extraction with rule editing

  4. Natural language processing

    Analyze natural language sentences to extract desired data

  5. Data refinement

    Refine data based on user domains

The Hancom OCR SDK recognizes approximately 99.7% of English documents, detecting text and extracting data from various unstructured formats, such as general documents, tables, and handwriting.

  • Table documents

  • Identification cards

  • Handwriting documents

  • On-screen characters

  • Curved text
